==와 ===의 차이

  1. == (equality operator)

    2. 자동 유형변화 비교
    서로 다른 유형의 두 변수의 "값" 비교

  2. === (strict equality operator 또는 identity operator)

    두 피연산자의 유형도 비교

alert(1 == 1); true
alert(1 == 2); false
alert(1 == "1"); true
alert(1 === 1); true
alert(1 === "1"); false
alert(0 == "0"); true
alert(0 == []); true
alert("0" == []); false
alert(null == undefined); true
alert(null === undefined); false
alert(true == 1); true
alert(true === 1); false
alert(true == "1"); true
alert(true === "1"); false
alert(0 === -0); true
alert(NaN === NaN); false

Javascript에서는 ===를 사용하는 습관을 들이자.
!= 연산자와 !== 연산자도 있다.
