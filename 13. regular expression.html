정규표현식

정규표현식은 const pattern = /a/; 와 같은 방식으로 사용.

pattern.exec(str);

str 안에 pattern이 있다면 해당 문자(열)을, 없다면 null 출력

pattern.test(str);

str 안에 pattern에 해당하는 문자가 있다면 true, 없으면 false 반환

str.match(pattern);

≒ pattern.exec(str);

str.replace(pattern, "A");

str에서 pattern에 해당하는 문자열을 "A"로 변환한다.

const pattern = /a/ig;

i는 대소문자를 구별하지 않는다. g는 모든 검색결과를 출력한다.
다만, g가 pattern.exec(str)에서는 적용 x, str.match(pattern)에서는 적용 o
